Gulp is awesome

Yes, its is!

  1. To get started with Gulp, I need to create a gulpfile.js
  2. In our gulpfile.js, we need to require a few libraries:
  3. gulp, gulp-uglify, gulp-contact, and gulp-watch
  4. Gulp is for runnig taks
  5. Concat is for concatenation
  6. Watch will listen for changes to your code and auto-run the tasks when changes are made
  7. Now, we need to install our libraries.. so far each of them in stall with --save-dev. Example: npm install gulp --save-dev
  8. We then need to define settings. Check the gulpfile for an example.
  9. After that, I create my first gulp.taks named default.
  10. Gulp requires that there is a default tasks.
